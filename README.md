# 🍦 Ice Cream Sales Prediction Using Polynomial Regression

- Developed a Polynomial Regression model to predict ice cream sales from temperature data exhibiting a non-linear relationship.
Compared Linear Regression and Polynomial Regression models using R² Score, MAE, and RMSE to evaluate performance.
Improved predictive accuracy through polynomial feature engineering and identified optimal model complexity by comparing multiple polynomial degrees.
Portfolio Rating

## Project Overview

- This project predicts ice cream sales based on temperature data. Initial analysis showed a non-linear relationship between    temperature and sales, making Linear Regression unsuitable. Polynomial Regression was applied to capture the curved pattern and improve predictive performance.

## Model Evaluation

- Polynomial Feature Engineering
- Degree Comparison
- Performance Evaluation
- Models Compared
- Linear Regression
- Polynomial Regression (Degree 2–6)
- Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## Results

The Linear Regression model underperformed due to the non-linear nature of the dataset. Polynomial Regression significantly improved predictive accuracy, with lower prediction errors and higher R² scores. Degree 2 and Degree 3 models provided the best balance between training and testing performance.
